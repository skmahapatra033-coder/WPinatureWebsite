## 📌 Project Overview

This project is a fully responsive **WordPress-based business website** developed for a **Food Product Manufacturing company in India**. The objective was to build a clean, modern, and conversion-focused website that showcases products, builds trust, and generates customer inquiries.

The website combines **custom development with WordPress flexibility**, ensuring scalability, performance, and easy content management for the client.

---

## 🚀 Key Features

* ✅ Custom WordPress Theme Development (Parent + Child Architecture)
* ✅ Elementor-based UI Design for flexible page building
* ✅ Fully Mobile Responsive Design
* ✅ Product Showcase with structured layouts
* ✅ SEO-Friendly Structure and Clean Code
* ✅ Fast Loading & Performance Optimized
* ✅ Contact & Inquiry Forms with Custom Integration
* ✅ Secure Email Delivery using PHPMailer (Custom Implementation)
* ✅ Easy Admin Panel for Content Management

---

## 🛠️ Tech Stack

* **CMS:** WordPress
* **Frontend:** HTML5, CSS3, Bootstrap
* **Builder:** Elementor
* **Backend:** PHP
* **Database:** MySQL
* **Email System:** PHPMailer (Custom-coded integration)
* **JavaScript:** jQuery / Vanilla JS

## 🧩 Development Approach

### 🔹 Custom Theme Architecture

The website uses a **Parent-Child Theme Structure** to ensure:

* Safe updates without breaking customizations
* Better maintainability
* Scalable development workflow

### 🔹 UI/UX Design

* Designed using Elementor for flexibility
* Clean and modern layout aligned with food industry branding
* Focus on readability, product visibility, and user engagement

### 🔹 Custom Functionality

* Integrated **custom PHP-based email system using PHPMailer**

## ⚙️ Setup Instructions

1. Install a fresh WordPress setup
2. Upload theme files to:
   `/wp-content/themes/`
3. Activate the **Child Theme**
4. Install required plugins (Elementor, etc.)
5. Configure database in `wp-config.php`
6. Set up email configuration for PHPMailer
7. Import demo content (if available)
